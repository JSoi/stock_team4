## 실시간 주가 현황 확인 어플리케이션

E반 4조의 첫번째 프로젝트!


### Description

- 비회원이 메인페이지를 접근하면 상위 4개의 주식 종류가 보인다.
- 회원이 메인페이지를 접근하면 자신이 등록한 주식목록과 관심있는 주식을 등록할 수 있다.

### Environment

- AWS EC2 (Ubuntu)

### Prerequisite

 1. Python 3.8
 2. mongoDB
 3. Flask, pymongo, dnspython, PyJWT, bs4...
